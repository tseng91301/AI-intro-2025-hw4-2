# 《Starseed Protocol 星辰種子計畫》
## 玩法介紹
### 操作面板
此遊戲是透過 console 頁面來與玩家互動，每次遊戲回合都會顯示一次操控面板，以下為操控面板範例:
```
--- Control Panel ---
資產: $1500
目前探索到的星球:
 - Earth (id=0) 資源=2 發現時的時間=2025-11-19 21:11:51

可執行的動作：
 1) 處理星球上的在地問題（從清單中選擇星球）
 2) 探索新星球（花費會隨回合增加）
 3) 將資源運送回地球
 4) 儲存遊戲
 5) 載入遊戲
 6) 離開遊戲
請選擇動作編號：
```
可以循著上面的操作提示(例如上面的畫面可以打 1~6 的數字來推進遊戲，若沒有特別提示則是按下 Enter 鍵完成輸入，例如 [2 -\> Enter] 探索新星球)
### 遊戲參數內容
#### 資產
用來探索星球所使用的籌碼，會以 `$金額` 計算，且值為 `int`
#### 資源
大部分星球會有的一些能夠增加資產的東西，會隨時間增加，有一個單位價值，以 `$金額` 計算，且值為 `int`
* 資源每 15 秒會增加一次，每增加一次會加一個資源價值
* 星球不能囤放太多資源，因此當時間達到 3 分鐘 (180 秒) 時，資源會停止累加，直到資源被運送回地球
* 關於 "資源運送" ，請參閱**主要故事線**的部分
### 主要故事線
#### 探索星球
花費億些資產來發現新的星球，如果星球上面有資源，還可以帶些戰利品回來。
* 每次探索星球的花費都會隨著目前已有的星球數量增加，公式是 `2000 + (探索星球數量 - 1)^1.7 * 350`，也可以透過修改程式 `explore_new_planet()` 更改算法
* 若資產不足以負荷當次探索的開銷，則無法探索新星球
#### 運送資源
* 玩家可以將在其他星球(包含地球在內)所採集的資產送回地球
* 每種資源在送回來地球時會需要一筆運費，不管最後能夠拿到多少盈利，只要資產 \< 運費就無法運送
* 若成功運送回地球，資產會 + 累計的此種資源價值 - 運費 (若太早運輸會造成負收益)
#### 處理星球問題
* 玩家可以去處理每一種星球上的問題，提供一個具體的解決方案，遊戲內核會評斷此解決方法的分數，並且給出收益的增減值(進行此任務可能賺也可能賠，但有機會拿到一大筆收益)

## 初次體驗遊戲
### 安裝關依賴套件包
1. 在您的 python 環境中輸入以下指令:
```shell
pip install -r requirements.txt
```
### 設定 openAI 的 API KEY
1. 在遊戲資料夾底下新增 config.json
2. 在 config.json 裡面寫入 api secret 資訊
```json
{
    "secret-key": "sk-proj-xxxxxxxxxxxxxxxxx"
}
```
### 重設遊戲進度 (可選)
為了遊戲體驗，建議刪除舊的遊戲存檔以利時間的同步，優化遊戲體驗。
1. 刪除既有的 `saves/starseed_save.json`
